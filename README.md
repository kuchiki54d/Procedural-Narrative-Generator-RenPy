# Procedural-Narrative-Generator-RenPy
A Python-based procedural content generation (PCG) engine for Ren'Py, dynamically creating narrative "books" from randomized datasets.

⚙PCG Algorithm:

Uses a tag-based filtering system to generate coherent story arcs (BL/GL/Saga) from external Python lists.

⚙Data-Driven Design:

The story database is fully decoupled from the execution logic, allowing for easy expansion and localization.

⚙Localization Demo:

This repository features an English dataset to showcase the system's adaptability to different languages.

⚙Maze Integration:

Includes a procedural maze navigation module that triggers random narrative events based on player movement.
